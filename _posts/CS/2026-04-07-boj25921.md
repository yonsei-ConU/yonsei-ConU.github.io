# 문제 이해하기
최소 스패닝 트리 지식이 필요한 문제는 아니지만, 음식의 가격이 두 정점을 잇는 가중치라고 생각하면 완전그래프에서 최소 스패닝 트리를 구하는 문제로 이해할 수 있습니다.
물론 이렇게 이해했다고 해서 문제풀이에 크게 도움이 되지는 않습니다만, 최소한 cyclefree여야 한다는 것 정도는 쉽게 알아낼 수 있지 않은가요? 아님 말고...

# 문제 풀기
우선 2의 배수 번호인 사람 전부를 잇는다고 생각해 봅시다.  
2와 2k는 gcd가 2이므로, 2와 4, 2와 6, ... 이렇게 이어줍니다.  
이렇게 하면 한 쌍당 무조건 2원이 듭니다.  
n의 배수로 일반화한다 해도 비슷한 방식으로 이어 주면 쌍당 n원이 듭니다.  
2의 배수끼리 이을 때 2번과 6번을 잇게 되고, 3의 배수끼리 이을 때 3번과 6번을 잇게 됩니다.  
그러면 그 중 3번과 6번을 잇는 걸 없애는 것이 더 이득입니다.  
이를 일반화하면, 소인수가 여러 개인 수의 경우 최소소인수만 남기면 됩니다.  
그렇게 하면 2번과 이어지는 번호는 4, 6, 8, 10, ... 그대로지만 3번과 이어지는 번호는 9, 15, 21 등이 될 것입니다.  
이렇게 하면 여러 개의 chain이 생기지만, 1번 그리고 N/2보다 큰 소수 번호 사람은 이어지지 않습니다.  
이 사람들도 이어 주기 위해 1번과 2번, 1번과 3번, 1번과 5번, ... 의 방식으로 1번과 각 소수를 이어 줄 것입니다.  

정리해봅시다.  
합성수 번호들은 자신의 최소 소인수 번호와 이어 놓았습니다.  
비용은 gcd였으므로 합성수 번호들의 비용 총합은 각각의 합성수에 대해 최소 소인수의 합입니다.  
모든 소수 번호들은 1번과 이어 놓았습니다.  
각 소수와 1번을 연결하는 비용은 그 소수 자체이며 소수의 최소 소인수는 자기 자신이므로 소수 번호 또한 각각의 소수에 대해 최소 소인수의 합입니다.  
결론적으로, 답은 2부터 N까지의 모든 자연수에 대해 자신의 최소소인수를 전부 더한 것입니다.  
이는 에라토스테네스의 체의 유명한 변형으로, 안 지워진 수를 지울 때 true를 false로 바꾸는 대신 어떤 소수가 자신을 지웠는지를 기록하면 됩니다.  
어차피 체는 가장 작은 수부터 확인하므로 맨 처음으로 자신을 지우는 수는 최소 소인수입니다.

# 코드
```py
N = int(input())
lpf = [0] * (N + 1)
for i in range(2, N + 1):
    if not lpf[i]:
        for j in range(i, N + 1, i):
            if not lpf[j]:
                lpf[j] = i
print(sum(lpf))
```
