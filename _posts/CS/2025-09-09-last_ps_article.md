# SUAPC 2025S

## 대회 후기
저번 글에서 SUAPC 2025S에 대해 자세히 언급하지 못했습니다. 당연하게도 그 이유는 오픈 콘테스트 진행 전이었기 때문에 문제 관련 얘기를 일절 할 수 없었기 때문입니다.  
SUAPC 2025S의 난이도와 태그 데이터를 모았습니다. 정렬 기준은 1: 난이도순, 2: 문제 번호순 입니다.  
A. B4 - math, arithmetic  
D. S2 - math, euclidean  
E. G5 - ad_hoc, binary search  
G. G1 - Knapsack DP, graph theory  
M. G1 - ad_hoc, difference array  
I. P5 - ad_hoc, game theory  
K. P5 - ad_hoc, primality test  
F. P3 - tree, constructive  
H. D5 - math, ad_hoc  
L. D5 - segment tree, case work  
C. D4 - ad_hoc, constructive  
J. D4 - Dijkstra, dominator tree  
B. D3 - 애드혹은 아닌데 뭐가 굉장히 많음  
총 13문제가 출제되었고, 그 중 7문제에 ad_hoc 또는 constructive 태그가 붙어있습니다. ad_hoc 태그가 붙은 태그는 6문제입니다.  
가능한 한 최악으로 본 대회에서 다른 블로그들처럼 몇 분에 누가 뭘 했고… 를 나열하는 건 별로 의미가 없을 것 같고 대회가 끝난 지 2주가 넘은 지금 시점에서 제대로 기억도 나지 않습니다. 그래서 문제별로 후기를 작성해 보겠습니다.  
저 말고 팀원분들 아이디가 한 분은 c, 한 분은 r로 시작하므로 c, r으로만 표기하겠습니다. 물론 저 후기들이 시간순이 아니므로 읽을 때 괴리감이 있을 수는 있어 보이네요…  
**A**. SUAPC 대회는 A번 문제는 쉽게 내고 나머지 문제들의 난이도 분포는 랜덤으로 잡는다고 알고 있습니다. 이번에도 A번은 쉽게 출제되었습니다. 문제 출제자님 팬이에요  
**D**. 잘못됨을 사실 여기서부터 느꼈습니다. C번을 조건 하나를 못 봐서 쉬운 문제인 줄 알고 풀이를 냈다가 WA를 받은 직후 D번으로 옮겨왔는데, (N+K)(1/A + 1/B + 1/C)=N이어야 한다는 거 하나면 다 되는 줄 알고 상당히 많은 시간을 소비했습니다. 어렵지 않은 문제인 것은 알고 있었지만 밀려나는 스코어보드 등수를 보고 대회 초반부부터 초조해지기 시작했고 r씨가 결국에 잘못된 점을 찾아 주어서 고치긴 했지만 이미 다들 이 문제를 풀고 난 뒤였습니다… 근데 제가 이걸 계속 봤으면 시간이 더 지나도 못 찾았을 만한 오류인 것 같습니다. 대대대… 아무튼 풀이시간은 약 1시간 30분…
**E**. 솔직히 아직도 이 문제를 왜 맞았는지 잘 모릅니다. 한 번 WA를 받고 다음 제출에서 AC를 받은 문제입니다. 두 코드는 이분탐색의 종료조건이 조금 다른데, 코드 로직이 어차피 이분탐색 루프를 빠져나가기 전에 프로그램이 종료되지 않는다면 틀렸습니다를 받아야만 하는 코드인데 그게 달라짐으로 맞은 이유를 저는 아직도 모릅니다. 이걸 푼 시점도 무려 2시간 30분.  
**G**. 제가 유일하게 제대로 푼 문제가 이겁니다. 제 생각에는 다른 팀원들을 시켰으면 절대 못 풀었을 만한 문제지만 그래도 제가 잘 푸는 유형의 문제이며, 이 블로그가 SUAPC 해설 블로그가 아니기에 풀이를 적지는 않겠지만 나름 만족스러운 역추적 냅색 풀이를 완성시킬 수 있었습니다. 이걸 푼 시점은 3시간 20분 정도네요.
**M**. 팀원 세 명 다 이 문제를 가지고 어느 정도는 고민했었는데, 씹덕자구 문제인지 애드혹인지조차 결론이 안났습니다. 저는 스코어보드 때문에라도 애드혹이라고 생각했는데 역시나 애드혹이었고, 뭐 골1이긴 하지만 매우 풀기 싫게 생긴 문제라 얘에 대해서는 그나마 미련이 없습니다.  
**I**: 저는 스프라그-그런디 정리를 사용해 그런디 테이블을 채우려다가 실패했고, 팀원 분들은 애드 혹 관점에서 문제를 풀어내려고 시도했지만 실패했던 문제입니다. 제가 자세히 봤던 문제도 아니고 맞힌 문제도 아니라 할 말이 많지는 않습니다.  
**K**. c씨가 남은 문제 중에 그나마 이거라도 풀어야겠다고 생각했고 제 옆자리에서 굉장히 오랫동안 고민하는 것을 봤습니다. 제가 E를 풀고 G를 풀기 전쯤에 15분쯤 봤었는데 저는 전혀 모르겠더라고요. 하여튼 c씨가 코드를 짜더니 (저는 설명을 안 들어서 모르지만) 논리적으로 꽤 완벽한 코드가 나왔다고 하고, 제가 소인수분해 템플릿을 복붙해서 갖다 주니까 바로 AC를 받았습니다. 대대대...  
**F**. A, D, E, G, K번을 모두 풀고 남은 게 이거밖에 없다는 생각으로 도전했지만 그냥 죽었습니다. 상당히 깔끔하고 팀원들 아무도 논리적인 반박을 못했던 풀이인데 며칠 후 제 풀이대로 구성하면 출력이 트리가 아닐 수도 있게 된다는 사실을 알게 됐습니다.  
나머지 문제는 사실상 제대로 고민해볼 시간도 없었기 떄문에 적을 내용이 없습니다.  

## 대회 뒷이야기
저번 글에서 오류가 있었는데, 저희 팀 등수가 12등이 아니라 16등이었습니다.  
스코어보드를 까는 동안 대회장에 있던 친구가 야 너네 12등이야 이렇게 말해준 걸 가지고 내용을 적었었는데, 스코어보드 오픈이 완전히 끝나기 전에 잘못 말한 것이 아닐까 싶습니다.  
물론 글에서 언급되어 있듯 상을 못 받은 것 때문에 아쉬운 마음은 딱히 없기 때문에 12등이든 16등이든 크게 신경쓰이지는 않습니다.  
저 글을 쓸 때 쓰고 싶었지만 못 쓴 후기는 이 정도로 마무리지으면 좋을 것 같습니다.  
저곳에서는 이 대회의 애드 혹 성향이 얼마나 강했느냐와 같은 얘기를 자세히 풀 수 없었는데, 시간이 지나서 다시 정리하는데도 참 웃긴 셋이라는 생각이 듭니다.  
어쨌든 그냥 모든 문제에서 다 털렸습니다. 그간의 노력이 모두 물거품이 되는 것 같은 경험이었어요. 저번 글 마지막 부분에도 적었지만 “되는 것 같은” 이라는 말보다는 조금 더 확신에 찬 말투로 적어야 할 것 같지만요.  
제가 언급할 만한 다른 연대 팀들은 두 팀 정도입니다.  
물론 1등 팀도 연세대 팀이고 연세대 팀이 굉장히 많긴 하지만, 제가 팀 구성원들을 잘 알고 있는 팀은 많지 않습니다.  
7위의 **권현우팬클럽**은 할 말이 좀 있습니다.  
8월 초쯤에 저한테 갑자기 너 ps해?? 이거 대회 상탄거 너임?? 이러고 카톡 보내는 친구가 있길래 그 친구를 아는 디스코드방으로 납치했습니다.  
솔브드 스트릭으로 봤을 때 PS를 제대로 시작한 건 2025년 7월 정도인 것 같은데 그때부터 꽤 많은 문제를 풀더니 다이아 문제를 대회 중 솔브하고 있더라고요. 이게 뭔상황이지  
팀 내부에서 각 팀원이 얼마 정도 문제를 풀었는지 자세한 상황까지는 모르지만, 경험치에 비해 매우 높은 퍼포먼스를 기록하더라고요.  
17위의 **척척첨컴**은 연세대 25학번 친구들로만 구성된 팀입니다. 김x윤씨 블로그에서는 빠진 것 같네요 :(  
저는 이제 저번 SUAPC때 팀빨을 정말 강하게 받았고 (블루 오렌지 오렌지) 어느 정도 저와 잘 맞는 성향의 셋이 출제되어 운빨로 상을 탔는데, PS를 제대로 한 기간이 길지는 않다고 알고 있는 후배들끼리 나가서 이정도 풀고 왔다는 것도 저는 매우 대단하다고 생각합니다. 이 팀이 저희보다 페널티 관리를 훨씬 더 잘하고 있었어서 여기서 하나 더 풀었으면 따잇당할 뻔…

# 무재능
지금 언급하는 내용은 저번 글 내용과 일부 겹치는 것 같긴 합니다만…  
저도 분명 충분히 저만의 강점이 있다고 생각했습니다. 일단 타자가 매우 빠른 편이며 구현 속도 또한 티어가 비슷한 사람들 사이에서는 매우 빠른 편이고, 찐렌지 정도 되는 분들과 비교해도 속도만큼은 크게 지지 않습니다.  
그런데 애드 혹의 경우 대회 이후에 더 생각나는 게 여기는 진짜 제가 커버할 수 있는 영역이 애초에 아니었다는 생각이 자꾸 듭니다.  
1학년 2학기 때 송도 기숙사 지하에서 망령처럼 PS를 할 때 제가 입문시킨 친구 중에서 그쪽 문제를 놀라울 만큼 잘 푸는 친구가 하나 있습니다.  
걔를 봤을 때는 ~~다이아 애드혹 보고 그냥하면됨 발언만 안 했어도~~ 진짜 순수하게 저런 사람도 있구나 진짜 대단하다… 하고 넘어갔을 것 같습니다. 물론 다이아 애드혹 보고 그냥하면됨 발언을 했어도 대단한 건 마찬가지입니다.  
근데 SUAPC 전후로 더 많은 사람들을 만나보니, 그친구가 애드혹을 무진장 잘 푸는 것은 사실이지만 걔만 그런 건 아니며, 그런 애들이 생각보다 많으며 그런 애들이 열심히 한다면 실력이 놀랄 만큼 빠르게 오른다는 것도 알게 되었습니다.  
솔직히 전에도 다른 사람들으로부터 막 누구는 1년만에 레드 가기도 한다더라 같은 종류의 얘기를 듣더라도 별로 감흥이 없었습니다.  
1년 만에 레드를 갔다는 사람이 1년 동안 얼마나 열심히 했는지도 저는 모르는 일이고 저랑 관련있는 사람도 아니라서 크게 관심이 가지도 않았습니다.  
비슷한 맥락으로 누구는 08년생인데 레드를 갔다더라 이런 얘기를 들어도 딱히 감흥이 없었습니다.  
위와 같은 종류의 얘기를 꽤나 여러 번 들었지만 ~~우리나라 일인데도~~ 남의 나라 일 같고 그랬습니다.  
근데 저와 가장 가까운 사람들, 심지어 그중에는 ps를 제가 제대로 입문시킨 사람들도 있고, 아무튼 그 사람들보다 제가 문제를 몇 배는 더 많이 풀어봤고 몇 배는 ps를 오래 했고 대회도 몇 배는 많이 나가봤는데 이렇게 쉽게 따이는 걸 보고 정말 큰 회의감이 들었습니다.  
열심히 노력한 게 부정당하는 기분이라는 말을 두 글에 걸쳐서 몇 번 언급했던 것 같은데, 제가 느끼는 상황을 달리 표현할 더 좋은 단어나 문장이 딱히 떠오르지 않습니다.  
그냥 쟤네같은 사람이 ps 하지 제가 할 만한 게 아니라고 생각하고, ps판에도 소위 노력의 천재가 존재한다고 듣긴 했으나 제가 노력의 천재 소리를 들을 만큼 인생을 갈아넣으면서까지 알고리즘 공부에 쏟고 싶은 생각은 없고, 그냥 그렇습니다.  
제 친구들과 제가 입문시킨 친구들과 모르고리즘 선배님들로 구성된 디스코드 방이 하나 있습니다.  
거기서 예전에는 나만못해 같은 식으로 징징댔었습니다.  
물론 저는 제가 엄청난 알고리즘 고수라고 생각해본 적은 한 번도 없지만 진짜로 엄청 못한다고까지 생각해본 적은 많지 않았고, 그래서 온전히 진심을 다해 한 말은 아니었습니다.  
근데 요즘 그 디스코드방에서 여러 말 안 되는 성과들을 보고 나만못해 라기보다는 나만 무재능 소리가 절로 나옵니다.  
내가 쟤만큼 ps에 익숙했을 때 난 이만큼밖에 못했는데… 이런 종류의 생각이 계속 듭니다. SUAPC가 끝나고 ps를 안하기로 마음먹고 실제로도 안 하고 있을 때도요.  
진짜 진심으로 무재능인 것 같습니다…  
저보다 몇 분의 1만큼의 노력으로 저와 비슷하거나 오히려 더 뛰어난 실력을 가지게 되는 경우를 눈앞에서 보니까 이건 아닌 것 같다는 생각이 계속 듭니다.  

# 결론
네, 솔직히 열등감에 찌들어서 쓴 글 맞고, 보는 사람에 따라 거부감을 불러일으킬 수도 있는 글인 것 같습니다.  
하지만 아무튼 할 말은 다 하고 가겠습니다.
